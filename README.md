# Интерактивное меню кафе «Уют» — V1

Учебный проект: просмотр меню с фильтрами, карточки блюд и бронирование столика.

## Структура файлов

```
cafe-menu/
├── src/
│   ├── menu.json   — данные блюд (добавляй свои сюда)
│   ├── App.jsx     — все компоненты React
│   ├── App.css     — все стили
│   ├── index.css   — глобальный сброс
│   └── main.jsx    — точка входа (не трогай)
├── index.html
└── package.json
```

## Запуск

```bash
# 1. Перейди в папку проекта
cd cafe-menu

# 2. Установи зависимости (только один раз)
npm install

# 3. Запусти сервер разработки
npm run dev
```

После этого открой в браузере адрес, который покажет терминал — обычно http://localhost:5173

## Что умеет V1

- Просмотр меню с фильтрацией по категориям
- Открытие карточки блюда (фото, описание, вес, цена)
- Бронирование столика: имя, телефон, дата, время, количество гостей, зал для курящих / некурящих
- Адаптивная вёрстка (работает на телефоне)

## Как добавить новое блюдо

Открой `src/menu.json` и добавь объект по образцу:

```json
{
  "id": 13,
  "name": "Название блюда",
  "category": "Горячее",
  "price": 400,
  "weight": "300 г",
  "description": "Описание блюда.",
  "photo": "https://picsum.photos/seed/любое-слово/400/300"
}
```

Категория подхватится автоматически — кнопка фильтра появится сама.
